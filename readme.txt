Kurtan game source code restoration (DOS, EGA, BGI).
When compiled, produces byte-exact executable.
Requires original fontr.krt, vol0r.krt, vol1r.krt, vol2.krt and vol3r.krt 
files to run.

Functions and variables names were recovered thanks to unstripped debug info.
Use c.bat to compile.
Borland Turbo C 2.01 is expected to be found in ..\TC201 folder.
